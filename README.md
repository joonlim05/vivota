# Vivota Education (Tuition Business Website)
The digital touchpoint for a professional tuition centre, that helps students and parents explore the courses offered, get information about the tutors, and contact the tuition centre easily. This project is a front-end implementation of the website, built with modern web technologies such as NextJS and Tailwind CSS.


## Features
- Browse available courses and tuition schedule
- Contact tutors directly
- Fully responsive design for desktop and mobile
- Modern and user-friendly interface
